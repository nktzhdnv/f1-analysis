# Анализ Гонок F1

# Техническое задание: F1 Analytics Dashboard

## 1. Цель проекта
Разработать веб-дашборд для анализа и визуализации данных Формулы 1 с возможностью сравнения гонщиков и анализа стратегий.

## 2. Функциональные требования

### 2.1. Обязательный функционал (MVP):
- [ ] Отображение графиков трасс и гонщиков на ней
- [ ] Отображение результатов последней гонки (таблица)
- [ ] График изменения позиций по ходу гонки
- [ ] Визуализация стратегий пит-стопов топ-5 гонщиков
- [ ] Сравнение времени кругов 2-х выбранных гонщиков

### 2.2. Дополнительный функционал:
- [ ] Выбор разных гонок/сезонов
- [ ] Анализ телеметрии (скорость на трассе)
- [ ] Статистика по командам

## 3. Технические требования

### 3.1. Технологический стек:
- Backend: Flask
- Анализ данных: Pandas, NumPy
- Визуализация: Matplotlib
- Данные: Fast-F1 API
- Фронтенд: HTML/CSS + Bootstrap
